## Repository Layout
This repository is laid out as follows..

| Folder        | Description |
| ------------- |:-------------| 
| .secplugs     | contains the meta data to import into the Secplugs portal |
| scripts       | core scripts that use the Secplugs platform      |
| build         | build, package and publish scripts  |
