Power Automate

What is the Power Automate Plugin?

This plugin provides the connector and templates needed to send documents and links for analysis in your Microsoft Power Automate work flows.

How does it work?

The plugin provides working templates ready for you to import and use.

How do I get started?

Download the templates and follow the instructions to get it installed and then take a look at the examples.

To use additional features and the privacy of your own account, after registering in Secplugs.com, login with your username and create an API key to use with the scripts. Replace the key in the samples or create new scripts using these as an example. Use can then use the Secplugs console to view activity, run reports and do deeper retrospective threat analysis.
